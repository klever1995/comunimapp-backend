<p><strong>Tabla de Arquitectura y Tecnologías (Backend)</strong></p>

| Componente | Tecnología | Propósito |
| :--- | :--- | :--- |
| **Framework API** | FastAPI (Python) | Backend principal con validación automática |
| **Base de Datos** | Firebase Firestore | Almacenamiento NoSQL en la nube |
| **Autenticación** | Firebase Auth + JWT | Login, registro y gestión de sesiones |
| **Notificaciones** | Firebase Cloud Messaging (FCM) | Notificaciones push en tiempo real |
| **Almacenamiento** | Cloudinary | Gestión de imágenes y multimedia |
| **Análisis de Datos** | Google Gemini API | Generación de reportes ejecutivos con IA |
| **Métricas** | Python + Cachetools | Dashboard con KPIs y gráficas |
| **Email** | SendGrid | Envío de correos de verificación |
| **Plataforma de Despliegue** | Render | Hosting y despliegue en la nube |
